인증

거의 대부분의 API는 JWT 토큰을 이용한 인증을 요구합니다.

JWT 토큰은 로그인을 통해 발급 받을 수 있습니다.

토큰 없이 API 호출시 아무것도 작동하지 않습니다.
HTTP/1.1 200 OK

{
    "success": true,
    "message": "OK",
    "resultData": "..."
}
HTTP/1.1 401 Unauthorized

{
    "success": false,
    "message": "AUTH_REQURED",
    "resultData": null
}
